## 描述

牛牛正在编写Python代码，他有些纠结到底该选vscode还是pycharm，于是他决定投硬币决定，正面使用vscode，反面则使用pycharm。请你根据输入的01数字表示正反面，使用三元表达式if-else输出牛牛使用的哪种编译器。

### 输入描述：

输入一个数字0或者1，0表示硬币反面，1表示硬币正面。

### 输出描述：

输出vscode或者pycharm。

## 示例1

```
输入：

1

输出：

vscode

```

```python
x = int(input())

print('vscode') if x == 1 else print('pycharm')
```
