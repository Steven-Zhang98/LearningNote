## 描述

牛牛有最好的朋友列表['NiuMei', 'NiuKele', 'NiuNeng']，以及他们与自己在牛客网的刷题量列表[3, 4, 5, 8]，请你使用zip函数将这两个列表打包为一个列表。（舍弃最末尾牛牛的刷题量）

### 输入描述：

无

### 输出描述：

直接输出打包后的列表。

## 答案

```python 
list1 = ['NiuMei','NiuKele','NiuNeng']

list2 = [3,4,5,8]

list2.pop()

list3 = list(zip(list1,list2))

print(list3)
```
